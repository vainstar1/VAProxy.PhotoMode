# VA Proxy PhotoMode
Simple mod for photo mode in VA Proxy. Some code was taken from tairasoul for the freecam

# Keybinds
- Shift for 4x speed
- Scroll wheel up / down to go faster or slower
- Alt for 0.25x speed
- Spacebar to go up / Ctrl to go down
- Q / E to rotate camera left / right on the Z axis, R to reset
